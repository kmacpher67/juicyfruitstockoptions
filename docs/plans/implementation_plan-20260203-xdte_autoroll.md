# Implementation Plan - X-DTE Options UI & Auto-Roll Evaluation

## Goal Description
Implement the User Interface for the "Options Due in X Days" signal and the "Auto-Roll Evaluation" feature. 
The backend signals (expiring options) are already generated by `ExpirationScanner`. This plan focuses on surfacing these signals in the Dashboard and providing an interactive "Smart Roll" analysis tool to help the user decide whether to Roll, Hold, or Close.

## User Review Required
> [!NOTE]
> This feature relies on the existing `JuicyOpportunity` persistence. We will be filtering for `trigger_source="ExpirationScanner"`.
